Project Components

iris_agent:
    Basic keylogger and device screenshot tool. Captured data is sent to Discord.

iris_eye:
    Captures webcam snapshots and sends them to Discord.

iris_logger:
    Basic keylogger that sends captured keystrokes to Discord.

iris_shadow.apk:
    Android application that captures camera photos (new only) and sends them to Discord.

iris-pega:
Advanced data collection tool that:

    Uploads browser history (Google Chrome, Firefox, Microsoft Edge)

    Captures keystrokes

    Takes device screenshots

    Collects device name
    All data is sent to Discord.

kira_decipher_key:
    Utility tool used to decode keylogs generated by iris-pega.
